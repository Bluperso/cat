# cat

# Реализация команды `cat`


## Что может?
Поддерживает следующие флаги:
- -b (GNU: --number-nonblank) - нумеровать непустые строки вывода
- -n (GNU: --number) - нумеровать все строки вывода
- -s (GNU: --squeeze-blank) - подавлять повторяющиеся пустые строки вывода
- -e - эквивалентно -vE
- -E - отображать $ в конце каждой строки
- -t - эквивалентно -vT
- -T - отображать символы табуляции как ^I
- -v - отображать непечатные символы (кроме табуляции и перевода строки)
- Поддерживает работу с несколькими входными файлами и стандартным вводом.
- Выводит сообщения об ошибках (идентичные оригинальной утилите cat) в stderr.
- Имеется набор тестов с автоматическим сравнением результатов с системной утилитой cat.

### Что нужно?
Наличие GCC compiler


### Сборка и тестирование
- make - компиляция исполняемого файла
- make test - запуск тестов
- make clean - удаление всех созданных файлов при запуске команд выше
